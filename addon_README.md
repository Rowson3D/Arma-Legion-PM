A custom pause menu addon for **Arma Reforger**, built for the Legion community server.

## Features

- Legion branding and logo
- Server rules displayed in-game
- Community tagline and Discord link
- Custom hero panel with unit information

## Installation

Place this addon folder inside your Arma Reforger Workbench addons directory:
```
Documents\My Games\ArmaReforgerWorkbench\addons\Legion PM\
```
Then load `addon.gproj` in Workbench and compile assets.
