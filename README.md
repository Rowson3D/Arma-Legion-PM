## Getting Started

### Cloning with Submodules

```bash
git clone --recurse-submodules https://github.com/Rowson3D/Arma-Reforger-Scripts-and-Addons.git
```

If you already cloned without submodules:

```bash
git submodule update --init --recursive
```

### Keeping Submodules Up to Date

```bash
git submodule update --remote
```

## Requirements

- [Arma Reforger](https://store.steampowered.com/app/1874900/Arma_Reforger/)
- [Arma Reforger Workbench](https://community.bistudio.com/wiki/Arma_Reforger:Tools)

---

## Usage

1. Clone this repository into your Arma Reforger Workbench addons directory:
   ```
   Documents\My Games\ArmaReforgerWorkbench\addons\
   ```
2. Open Workbench and load the relevant `.gproj` file from the addon you want to work with.
3. Compile assets as needed via the Resource Browser.
